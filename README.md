# Arac-Kiralama-Sistemi
Kastamonu Üniversitesi Programlama II dersi için OOP prensipleriyle geliştirilmiş modüler Araç Kiralama Otomasyonu.
Lüks Araç Kiralama Otomasyonu (v1.0)

Bu proje, modern yazılım mimarileri kullanılarak geliştirilmiş profesyonel bir Masaüstü Araç Kiralama Otomasyonu çözümüdür. Proje, sadece bir ödev değil; veri yönetimi, kullanıcı deneyimi ve nesne tabanlı programlama prensiplerinin birleşimidir.

🛠 Kullanılan Teknolojiler Dil: Python 3.x

Arayüz (GUI): Tkinter (Standart Python GUI Kütüphanesi)

Veritabanı: SQLite3 (İlişkisel Veri Yönetimi)

Veri Kaydı: JSON (Hafif ve Hızlı İşlem Kayıtları)

Mimari: OOP (Nesne Tabanlı Programlama) & Modüler Yapı

🌟 Öne Çıkan Özellikler Dinamik Veri Yönetimi: Araç listesi kodun içinde değil, profesyonel bir SQL veritabanında tutulur.

Kalıcı İşlem Kaydı: Yapılan her kiralama işlemi, sistem kapansa bile gecmis.json dosyasına kalıcı olarak işlenir.

Nesne Tabanlı Mimari: Tüm araçlar ve işlemler sınıflar (Classes) üzerinden yönetilir; bu sayede kod ölçeklenebilir ve geliştirilebilir.

Kullanıcı Dostu Arayüz: Karmaşık SQL sorguları arka planda soyutlanmış (Abstraction), kullanıcıya basit ve etkili bir tablo yapısı sunulmuştur.

📁 Proje Dosya Yapısı arayuz.py: Kullanıcı etkileşimi ve SQL görselleştirme.

tasitlar.py: Araç sınıfları, Kalıtım (Inheritance) ve Kapsülleme (Encapsulation).

isletme.py: Kiralama mantığı ve JSON veri işleme.

veritabani_kur.py: SQL veritabanı şemasının oluşturulması.
